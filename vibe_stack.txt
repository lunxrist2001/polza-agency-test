МОЙ ВАЙБКОД-СТЕК (Polza Agency тестовое)

IDE: Claude Code (CLI)
Модель:Claude Opus 4.7

ИНСТРУМЕНТЫ:
- Claude Code — генерация кода, поиск информации, написание скриптов
- WebSearch — поиск компаний, контактов, фактов
- WebFetch — парсинг сайтов компаний
- Pilot MCP — управление браузером (попытка автоматизации Google Таблиц)
- Python 3 — скрипт персонализации (personalize.py)
- Git — версионный контроль

ПОДХОД:
- Сбор базы: агенты Claude Code + WebSearch по отраслям (IT, финтех, логистика, консалтинг, маркетинг)
- Персонализация: Python-скрипт с кэшем, подстановка реальных фактов о компаниях
- Цепочка писем: написана вручную с использованием ИИ, с соблюдением лимита 120 слов

ФАЙЛЫ ПРОЕКТА:
- companies_final.csv — исходный список 54 компаний
- companies_personalized.csv — база с персонализацией
- polza_companies_final.csv — финальный файл для Google Таблиц (UTF-8 BOM)
- personalize.py — скрипт персонализации
- email_sequence.txt — цепочка из 3 писем
- vibe_stack.txt — этот файл
