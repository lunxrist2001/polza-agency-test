## 🎯 О ПРОЕКТЕ

Проект создан для **Polza Agency** — аутрич-агентства, специализирующегося на запуске email-маркетинга для B2B компаний.

**Задачи проекта:**
- Сбор базы из 54 целевых B2B компаний (IT, финтех, логистика, консалтинг, маркетинг)
- Автоматическая персонализация писем с реальными фактами о компаниях
- Формирование цепочки из 3 писем (холодное письмо + 2 фоллоу-апа)
- Подготовка финальной базы для импорта в Google Таблицы

## 🛠 СТЕК ТЕХНОЛОГИЙ

**ИНСТРУМЕНТЫ:**
- 🤖 **Claude Code (CLI)** — генерация кода, поиск информации, написание скриптов
- 🌐 **WebSearch** — поиск компаний, контактов, фактов
- 🔍 **WebFetch** — парсинг сайтов компаний
- 🐍 **Python 3** — скрипты обработки данных
- 📊 **CSV** — хранение базы компаний
- 🔧 **Git** — версионный контроль

**ПОДХОД:**
1. **Сбор базы:** агенты Claude Code + WebSearch по отраслям
2. **Персонализация:** Python-скрипт с кэшем, подстановка реальных фактов о компаниях
3. **Email-цепочка:** написана с использованием ИИ, соблюдение лимита 120 слов на письмо

## 🚀 КАК ИСПОЛЬЗОВАТЬ

### 1. Подготовка базы компаний
```bash
python personalize.py
```
Скрипт обрабатывает `companies_final.csv`, добавляет персонализированные факты и сохраняет в `companies_personalized.csv`.

### 2. Сборка финального CSV
```bash
python build_final_csv.py
```
Создает `polza_companies_final.csv` с BOM для корректного открытия в Excel/Google Таблицах.

### 3. Использование цепочки писем
Откройте `email_sequence.txt` — там готовая цепочка из 3 писем с переменными для подстановки:
- `{{название компании}}`
- `{{имя контакта}}`
- `{{персонализация}}`
