# Polza Agency Test

Тестовый репозиторий для Polza Agency.

## Описание

Этот проект содержит скрипты и данные для работы с компаниями, включая персонализацию и генерацию email-последовательностей.

## Структура проекта

- `build_final_csv.py` — скрипт для сборки финального CSV
- `companies_final.csv` — финальный список компаний
- `companies_personalized.csv` — персонализированные данные компаний
- `companies_raw.txt` — исходные данные компаний
- `email_sequence.txt` — последовательность email-сообщений
- `personalize.py` — скрипт для персонализации данных
- `polza_companies_final.csv` — итоговый CSV файл компаний Polza
- `vibe_stack.txt` — описание технологического стека

## Использование

Запуск скриптов:
```bash
python personalize.py
python build_final_csv.py
```
